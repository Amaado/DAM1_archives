Etiquetas usadas en tablas HTML
Etiqueta Descripción Atributos destacables
< table > Permite definir una tabla border
< tr > Declara una nueva línea de la tabla
< td > Define una celda dentro de una fila colspan, rowspan
< th > Define una celda dentro de la primera fila (cabecera)
< caption > Permite indicar el título de una tabla
< colgroup > Permite aplicar atributos a una o varias columnas span
< col > Permite indicar las columnas individualmente cuando se usa dentro de < colgroup >
< thead > Separa (estructuralmente) la primera fila de la tabla del resto
< tbody > Separa (estructuralmente) todas las filas de la tabla de la primera y última
< tfoot > Separa (estructuralmente) la última fila de a tabla del resto
